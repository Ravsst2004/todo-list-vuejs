# todo-app-vue

Simple Todo List Web using Vuejs. Use Some Library like Axios, Day.js, Json Server.

Need install Vite first
```
npm install vite
```

Run the project
```
npm run dev
```

Then run the fake json db server in another terminal
```
npx json-server --watch data/db.json --port 8000
```
